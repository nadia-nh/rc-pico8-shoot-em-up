# pico8-shoot-em-up
A shoot-em-up style tiny game using PICO-8

### To try it out
We need to load the contents of the game along with the assets in order to try out the game.
This shows how to use the free online educational edition of PICO-8.

- Go to https://www.pico-8-edu.com/
- Press `ESC` on the terminal to open the editor
- Copy and paste the contents of exported.p8
- Press `ESC` again to go back to the terminal
- Type `SAVE game.p8` (or choose a different filename)
- Type `LOAD game.p8`
- Type `RUN`
- Play the game!

On the desktop version of PICO-8, type `LOAD <PATH-TO-FILE>\exported.p8.png` and then `RUN`.
